## Developing

Clone the repository

```bash
git clone git@github.com:FawadHa1der/CairoBlackScholesUI.git
cd cairopal
```

Install Dependencies

```bash
yarn
```

Run locally

```bash
yarn dev
```

#### Run tests

```bash
yarn test
```
